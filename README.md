# Weather App 🌦️

A full-stack weather application built using Node.js, Express.js, and Visual Crossing Weather API.

## Features
- Live weather data
- Celsius temperature
- Redis caching
- Rate limiting
- Frontend UI

## Technologies Used
- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- Redis

## Run Project

npm install
node server.js

## API Endpoint

http://localhost:3000/api/weather/mumbai
